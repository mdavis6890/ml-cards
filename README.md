## 🚀 How to Import to Anki

This deck is provided in **CrowdAnki format**, which allows for easy updates and stable card management.

1.  **Install the CrowdAnki Add-on**:
    - Open Anki.
    - Go to `Tools` -> `Add-ons`.
    - Click `Get Add-ons` and enter the code: `1788670778`.
2.  **Import the Deck**:
    - Go to `File` -> `CrowdAnki: Import from directory`.
    - Select the `ai_flashcards/` folder in this repository.
    - Click `OK`.

## 🛠️ Card Structure

Each card follows a consistent model:
-   **Front**: The term, question, or concept.
-   **Back**: A concise definition or explanation.
-   **Reference Link**: A URL for deeper learning.
-   **Last Updated**: The date of the last verification (e.g., `2026-03-15`).
